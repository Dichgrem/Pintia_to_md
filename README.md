# Pintia 题目导出工具

从 Pintia 平台导出题目集为 Markdown 文件。

## 功能特性

- 支持选择题、编程题、SQL编程题、填空题等多种题型
- 自动提取用户提交的答案
- 可视化界面，一键导出（油猴脚本）
- 命令行交互式导出


## 方式一：油猴脚本（推荐）

### 安装

1. 安装浏览器扩展：**ScriptCat** 或 **Tampermonkey**
2. 复制 `src/pintia-export.user.js` 内容到扩展中
3. 保存脚本

### 使用

1. 访问 https://pintia.cn 并登录
2. 点击页面右下角 📤 按钮
3. Session ID 会自动获取，如需手动输入会弹出提示框
4. 点击题目集的"导出"按钮或"导出所有题目集"
5. 文件自动下载并复制到剪贴板

### 获取 Session ID

如需手动输入：
1. F12 打开开发者工具 → Network 标签 → 刷新页面
2. 找到任意请求 → Request Headers
3. 复制 `Cookie: PTASession=xxx` 中的 `xxx`

---

## 方式二：命令行脚本

### 安装

```bash
bun install
```

### 使用

```bash
bun run export
```

或设置环境变量：
```bash
export PINTIA_SESSION_ID="your_session_id"
bun run export
```
